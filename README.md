Aplicación de sitio de DropBox real dedicado al
almacenamiento de archivos en la nube.

Dentro de las funcionalidades de la app, están:
1. El proceso para autentificar y registrar un usuario ante el sistema se mantiene.
2. El usuario puede examinar el contenido de sus carpetas.
3. El usuario puede subir, descargar, borrar sus propios documentos.
